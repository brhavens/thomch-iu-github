  • An application, also referred to as an application program or application software, is a computer software package that performs a specific function directly for an end user or, in some cases, for another application. (Gillis 2022)​
    • Applications can be executed on a PC such as Word or Excel. They can also be executed on a server to run web or database functions.​
    • Applications can also be complex, enterprise-wide solutions such as an Enterprise Resource Planning package (SAP)​
