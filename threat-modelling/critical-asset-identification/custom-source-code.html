--- layout: iu-template title: "Custom Source Code" ---
  • Custom source code is computer code written for a specific purpose or function. Organizations often write code specific to their products and services. ​
    • Custom source code can be written to build an application specifically designed to accept IP based payment transactions.​
    • Custom source code could also be used to build front-end systems to process eCommerce transactions or back-end systems to settle the transactions.​
